= Acesse o console de vídeo ONTAP Select
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Você pode acessar o console de vídeo da máquina virtual do hipervisor onde o ONTAP Select está em execução.

.Sobre esta tarefa
Pode ser necessário acessar o console da máquina virtual para solucionar um problema ou quando solicitado pelo suporte da NetApp .

.Passos
. Acesse o cliente vSphere e faça login.
. Navegue até o local apropriado na hierarquia para localizar a máquina virtual ONTAP Select .
. Clique com o botão direito do mouse na máquina virtual e selecione *Abrir console*.
